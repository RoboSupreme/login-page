Classroom Setup Test

This page will test if your Supabase database is correctly set up for the classroom functionality:

  1. It will check if your user profile has the role field
  2. It will attempt to create a test classroom
  3. It will add your user as an admin to that classroom
  4. It will create a test journal entry in that classroom
  5. It will retrieve journal entries for that classroom

Important: You must be logged in first. Please make sure musicUsername is set in your localStorage.

Check your browser console (F12 or right-click → Inspect → Console) to see the test results.
